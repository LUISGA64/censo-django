# 🔍 Búsqueda Global - Guía Rápida

## 🎯 ¿Qué hace?

Busca **instantáneamente** en todo el sistema:
- 👥 Personas (por nombre, apellido o cédula)
- 🏠 Fichas familiares (por número)
- 📄 Documentos (por código o beneficiario)

---

## 🚀 ¿Cómo usarla?

### Opción 1: Desde la Barra Superior

```
┌────────────────────────────────────────────────────┐
│  CENSO WEB            [🔍 Buscar...]    👤 SALIR  │
└────────────────────────────────────────────────────┘
                         👆 Click aquí
```

1. Click en el campo de búsqueda
2. Escribir lo que buscas
3. Presionar **Enter**

### Opción 2: Desde el Menú

```
┌─────────────────────┐
│ 📊 Dashboard        │
│ 🔍 Búsqueda Global  │ 👈 Click aquí
│ 🏢 Asociación       │
│ 🏠 Ficha Familiar   │
└─────────────────────┘
```

## 🔎 Ejemplos de Búsqueda

### Buscar Personas

```bash
# Por nombre
juan          → Encuentra a Juan, Juana, Juanita...

# Por apellido  
garcía        → Encuentra todos los García

# Por cédula
123456789     → Encuentra persona con esa cédula

# Nombre completo
juan garcía   → Más específico
```

### Buscar Fichas

```bash
# Por número de ficha
1             → Ficha #1
10            → Ficha #10
100           → Ficha #100
```

### Buscar Documentos

```bash
# Por número de documento
DOC-2024-001  → Documento específico

# Por año
DOC-2024      → Todos los docs de 2024

# Por beneficiario
maría lópez   → Docs de María López
```

## ⚡ Características

### Búsqueda Inteligente
- ✅ No distingue mayúsculas/minúsculas
- ✅ Busca en partes de palabras
- ✅ Resultados instantáneos

### Seguridad
- 🔒 Solo ves datos de **tu organización**
- 👑 Superusuarios ven **todas** las organizaciones

### Límites
- 📊 Máximo **10 resultados** por categoría
- ⚠️ Mínimo **2 caracteres** para buscar

## 💡 Tips

### ✅ Búsquedas Efectivas

```bash
✅ juan          → Rápido, encuentra varios
✅ 123456        → Específico, encuentra uno
✅ garcía        → Apellido común
✅ DOC-2024      → Todos de 2024
```

### ❌ Evita Esto

```bash
❌ j            → Muy corto (mínimo 2 caracteres)
❌ a            → Muy general (demasiados resultados)
❌ juan garcía lópez → Muy específico
```

## 🛠️ ¿Problemas?

### No muestra resultados
- ✅ Verifica que escribiste mínimo 2 caracteres
- ✅ Intenta con términos más generales
- ✅ Verifica que hay datos en el sistema

### Resultados incompletos
- ℹ️ Se muestran máximo 10 resultados por tipo
- ℹ️ Sé más específico en tu búsqueda

### Muy lento
- ℹ️ Normal si hay muchos datos
- ℹ️ Usa términos más específicos
