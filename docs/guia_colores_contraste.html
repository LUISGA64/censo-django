Niveles de Conformidad WCAG

AAA (Mejorado)

Ratio mínimo: 7:1

El nivel más alto de accesibilidad. Recomendado para todas las aplicaciones.

AA (Estándar)

Ratio mínimo: 4.5:1

Nivel mínimo aceptable para cumplir con estándares de accesibilidad.

FAIL

Ratio menor a 4.5:1

No cumple con estándares de accesibilidad. NO usar para texto.
