## 💵 MODELOS DE NEGOCIO

### Modelo 1: SaaS (Software as a Service) - RECOMENDADO INICIAL

**Descripción:**
Sistema en la nube, pago mensual/anual.

**Ventajas:**
- ✅ Ingresos recurrentes
- ✅ Actualizaciones automáticas
- ✅ Soporte centralizado
- ✅ Menor barrera de entrada
- ✅ Escalabilidad

### Instalación Manual (Para desarrolladores)

**Script completo incluido en el paquete:**

```bash
#!/bin/bash
# Instalador censo-django on-premise
# Versión 1.0

echo "==================================="
echo "Censo Indígena - Instalador Local"
echo "==================================="

# 1. Actualizar sistema
apt update && apt upgrade -y

# 2. Instalar dependencias
apt install -y python3.10 python3-pip python3-venv
apt install -y postgresql postgresql-contrib
apt install -y nginx

# 3. Crear usuario y base de datos
sudo -u postgres psql << EOF
CREATE DATABASE censo_db;
CREATE USER censo_user WITH PASSWORD 'password_seguro';
ALTER ROLE censo_user SET client_encoding TO 'utf8';
ALTER ROLE censo_user SET default_transaction_isolation TO 'read committed';
ALTER ROLE censo_user SET timezone TO 'America/Bogota';
GRANT ALL PRIVILEGES ON DATABASE censo_db TO censo_user;
EOF

# 4. Crear directorio de aplicación
mkdir -p /opt/censo-django
cd /opt/censo-django

# 5. Descomprimir aplicación
tar -xzf /tmp/censo-django-package.tar.gz

# 6. Crear entorno virtual
python3 -m venv venv
source venv/bin/activate

# 7. Instalar dependencias Python
pip install -r requirements.txt

# 8. Configurar variables de entorno
cat > .env << EOF
SECRET_KEY=$(python3 -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())')
DEBUG=False
ALLOWED_HOSTS=localhost,127.0.0.1,$(hostname -I | awk '{print $1}')
DATABASE_NAME=censo_db
DATABASE_USER=censo_user
DATABASE_PASSWORD=password_seguro
DATABASE_HOST=localhost
DATABASE_PORT=5432
SITE_URL=http://localhost
EOF

# 9. Aplicar migraciones
python manage.py migrate

# 10. Crear superusuario
python manage.py createsuperuser --noinput \
    --username=admin \
    --email=admin@localhost

# 11. Recolectar archivos estáticos
python manage.py collectstatic --noinput

# 12. Configurar Gunicorn
cat > /etc/systemd/system/censo.service << EOF
[Unit]
Description=Censo Django
After=network.target

[Service]
User=www-data
Group=www-data
WorkingDirectory=/opt/censo-django
Environment="PATH=/opt/censo-django/venv/bin"
ExecStart=/opt/censo-django/venv/bin/gunicorn --workers 3 --bind unix:/opt/censo-django/censo.sock censoProject.wsgi:application

[Install]
WantedBy=multi-user.target
EOF

# 13. Configurar Nginx
cat > /etc/nginx/sites-available/censo << EOF
server {
    listen 80;
    server_name localhost;

    location = /favicon.ico { access_log off; log_not_found off; }
    
    location /static/ {
        root /opt/censo-django;
    }
    
    location /media/ {
        root /opt/censo-django;
    }

    location / {
        include proxy_params;
        proxy_pass http://unix:/opt/censo-django/censo.sock;
    }
}
EOF

ln -s /etc/nginx/sites-available/censo /etc/nginx/sites-enabled

# 14. Iniciar servicios
systemctl start censo
systemctl enable censo
systemctl restart nginx

# 15. Configurar firewall
ufw allow 80/tcp

echo "==================================="
echo "Instalación completada!"
echo "Acceder a: http://$(hostname -I | awk '{print $1}')"
echo "Usuario: admin"
echo "Contraseña: (ver archivo /opt/censo-django/admin_password.txt)"
echo "==================================="
```

## 💾 BACKUP Y RECUPERACIÓN

### Sistema de Backup Automático Incluido

**Script de backup diario:**
```bash
#!/bin/bash
# /opt/censo-django/scripts/backup.sh

BACKUP_DIR="/opt/censo-django/backups"
DATE=$(date +%Y%m%d_%H%M%S)

# Backup de base de datos
pg_dump -U censo_user censo_db | gzip > "$BACKUP_DIR/db_$DATE.sql.gz"

# Backup de archivos media
tar -czf "$BACKUP_DIR/media_$DATE.tar.gz" /opt/censo-django/media

# Mantener solo últimos 30 días
find $BACKUP_DIR -type f -mtime +30 -delete

echo "Backup completado: $DATE"
```

**Configuración en cron:**
```bash
# Ejecutar diariamente a las 2 AM
0 2 * * * /opt/censo-django/scripts/backup.sh
```
