## 🎨 GRADIENTES CORPORATIVOS

### Gradiente Primario (Azul)
```css
background: linear-gradient(135deg, #1E40AF 0%, #2563EB 100%);
```
**Uso:** Headers, botones primarios, navegación principal

### Gradiente Oscuro (Gris)
```css
background: linear-gradient(135deg, #1E293B 0%, #334155 100%);
```
**Uso:** Sidebar, fondos oscuros, footers

### Gradiente Éxito (Verde)
```css
background: linear-gradient(135deg, #047857 0%, #059669 100%);
```
**Uso:** Confirmaciones, aprobaciones, estados positivos

### Gradiente Info (Cyan)
```css
background: linear-gradient(135deg, #0284C7 0%, #0EA5E9 100%);
```
**Uso:** Información, tooltips, mensajes informativos

## 💫 SOMBRAS PROFESIONALES

### Sombra Pequeña
```css
box-shadow: 0 1px 2px 0 rgba(0, 0, 0, 0.05);
```
**Uso:** Separadores sutiles, elementos planos con ligero relieve

### Sombra Estándar
```css
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 
            0 2px 4px -1px rgba(0, 0, 0, 0.06);
```
**Uso:** Cards, botones elevados, dropdowns

### Sombra Grande
```css
box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 
            0 4px 6px -2px rgba(0, 0, 0, 0.05);
```
**Uso:** Modales, elementos flotantes, hover states

### Sombra Extra Grande
```css
box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1), 
            0 10px 10px -5px rgba(0, 0, 0, 0.04);
```
**Uso:** Popovers, menus contextuales, elementos prominentes

## 📐 ESPECIFICACIONES TÉCNICAS

### Contraste WCAG
- ✅ **AAA** para todos los textos principales
- ✅ **AA** mínimo para textos secundarios
- ✅ Ratio mínimo 7:1 para títulos
- ✅ Ratio mínimo 4.5:1 para texto normal

### Tipografía
```
Font Family: "Inter", -apple-system, BlinkMacSystemFont, "Segoe UI"
Font Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
Line Height: 1.6 (optimal para lectura)
Letter Spacing: -0.025em (títulos), 0.025em (botones)
```

### Border Radius
```
Small:  0.375rem (6px)
Normal: 0.5rem   (8px)  ⭐ Estándar
Large:  0.75rem  (12px)
XL:     1rem     (16px)
2XL:    1.5rem   (24px)
```

## 🎯 APLICACIONES RECOMENDADAS

### Headers y Títulos
```css
color: var(--neutral-900);
font-weight: 700;
background: linear-gradient(135deg, var(--primary-800), var(--primary-600));
```

### Botones Principales
```css
background: linear-gradient(135deg, var(--primary-800), var(--primary-600));
color: white;
box-shadow: 0 4px 6px -1px rgba(30, 64, 175, 0.3);
```

### Cards y Contenedores
```css
background: white;
border: 1px solid var(--neutral-200);
border-radius: 0.75rem;
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1);
```

### Texto Principal
```css
color: var(--neutral-700);
font-size: 0.9375rem;
line-height: 1.6;
```

### Enlaces
```css
color: var(--primary-600);
text-decoration: none;
font-weight: 500;
```

```css
/* Hover */
color: var(--primary-700);
text-decoration: underline;
```
